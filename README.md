# Bomberman

## Venv :
- windows:
  - py -3 -m venv .venv
  - .venv\Scripts\activate
- linux:
  - python3 -m venv venv
  - source venv/bin/activate

## Rodar Cliente :
- pip install -r requirements.txt
- python Client.py

## Rodar Server : 
O servidor não tem nenhuma dependencia fora a biblioteca padrão do python
- python Server.py
